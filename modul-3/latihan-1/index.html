Tahapan Menghidupkan Laptop

  1. Kabel power isi daya baterai sudah dihubungkan ke sumber listrik
  2. Menekan tombol power pada keyboard dekat layar atau bagian pinggir laptop
  3. Tunggu proses booting atau ikuti Langkah prosesnya
  4. Selesai dan laptop sudah siap untuk digunakan

Istilah Teknologi Terkait Komputer dan Internet

Access Point
Perangkat apa pun yang memungkinkan perangkat nirkabel terhubung ke jaringan kabel.
API
Seperangkat fungsi yang memungkinkan fitur perangkat lunak diakses oleh aplikasi.
Bandwith
Kecepatan transfer data maksimum dari koneksi atau jaringan internet.
Data center
Fasilitas yang menampung penyimpanan data, server, dan komputer.
DNS
Layanan nama domain, atau direktori alamat IP untuk nama domain web.
